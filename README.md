## Description
> This is **unofficial** [Odoo](https://www.odoo.com/) mobile client for Android with [HSE](https://www.hse.ru/en/) support. Enjoy!

## Features
- Authorization
  - General
  - HSE
- Selecting modules, pick your favorite
- Recruitment module `coming soon!`
- CRM module `coming soon!`
- Profile `coming soon!`
